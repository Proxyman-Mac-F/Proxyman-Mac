## Overview

Proxyman Professional Suite is a complete network debugging and testing platform that serves development teams with the traffic analysis depth, scripting capability, and collaboration tools needed for professional API development and quality assurance. It extends beyond passive inspection into active traffic manipulation and automated testing that handles the complex network debugging scenarios that modern application development produces. The platform supports debugging across macOS applications, iOS devices, simulators, and Android emulators.

Scripting engine enables programmatic request and response modification using JavaScript, allowing complex transformation logic, conditional routing, and automated test scenarios that manual breakpoints cannot handle. Scripts can modify any aspect of the request or response, inject delays, return mock responses, and log custom data. The scripting environment includes an editor with syntax highlighting and debugging output.

Diff tool compares any two captured requests or responses side by side with highlighted differences, enabling quick identification of changes between API versions, environments, or test runs. The comparison handles structured content formats with semantic diffing that identifies meaningful changes rather than formatting differences. This capability accelerates API regression testing and debugging.

Team collaboration through shared projects and exportable session files enables developers to capture network traffic, annotate interesting requests, and share the complete debugging context with team members without requiring them to reproduce the issue. Exported sessions preserve the full request and response data with annotations, making bug reports that include network context self-contained. Session files can be imported into any Proxyman installation for analysis.

Proxyman Professional Suite is the network debugging platform for development teams who need comprehensive HTTP/HTTPS inspection, traffic manipulation, and collaborative debugging in a tool that matches the quality of the applications they are building. Its native macOS design and modern interface make it the proxy tool that developers actually enjoy using.

## Performance & Integration

Proxyman is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Proxyman processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
